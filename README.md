# Expense
Checking the expenses using Angular through Standalone Components

This Angular Application is used to calculate the total expenses spent along with us being able to see the description, amount spent and date on which we spent.

As it is a standalone component main.ts has the bootstrapApplication which starts our Angular application and loads AppComponent, the Root Component which imports all the required components in the application and connect data.

AddExpenseComponent has input details of the form.

ExpenseListComponent has the list of expenses and 

ExpenseTotalComponent having the sum of all the expenses.
